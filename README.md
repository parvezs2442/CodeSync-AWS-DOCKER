# CodeSync-AWS-DOCKER 🚀

A production-ready real-time collaborative code editor with Dockerized full-stack architecture and AWS deployment support.

## 📌 Project Overview

CodeSync-AWS-DOCKER is a scalable collaborative coding platform where multiple users can join a room and code together in real time. It is designed using modern full-stack technologies with Docker containerization and cloud deployment readiness.

This project demonstrates practical skills in:

- Full Stack Development
- Real-time Communication
- Docker & DevOps
- AWS Deployment
- Scalable System Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js
- Socket.IO

### DevOps / Deployment
- Docker
- AWS EC2

## ✨ Key Features

✅ Real-time collaborative code editor

✅ Instant code synchronization  
✅ Fast React frontend with Vite
